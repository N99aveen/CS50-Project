## Description

The **Financial Literacy Assistant** is a Python-based Command Line Interface (CLI) tool designed to help users improve their understanding of personal finance. This program offers a wide range of functionalities, from learning financial terms to calculating savings, investments, loans, SIPs, and retirement planning.

It is designed as a **CS50P Final Project**, demonstrating Python programming skills including functions, user input handling, calculations, and modular programming.

---

## Features

1. **Learn Financial Terms**
   Users can explore 50+ financial terms such as Savings, Investment, SIP, EMI, Inflation, Mutual Funds, Bonds, Stocks, and more.

2. **Savings Calculator**
   Calculates monthly and yearly savings based on income and expenses, and provides savings habit analysis.

3. **Investment Calculator**
   Computes the future value of investments using compound interest.

4. **Loan & EMI Calculator**
   Calculates monthly EMI, total payment, and total interest for loans.

5. **SIP Calculator**
   Computes future value of systematic investment plans with given monthly contribution, rate of return, and duration.

6. **Finance Calculators**
   Includes:
   - Simple Interest Calculator
   - Compound Interest Calculator
   - CAGR Calculator
   - Inflation Adjusted Return Calculator

7. **Risk Profile Assessment**
   Helps determine if a user is a high, moderate, or low-risk investor based on a simple questionnaire.

8. **Budgeting Tips & Financial Health Check**
   Provides actionable budgeting advice and evaluates savings and debt ratio for financial health.

9. **Net Worth Calculator**
   Calculates net worth based on total assets and liabilities.

10. **Emergency Fund Calculator**
    Suggests recommended emergency fund based on monthly expenses.

11. **Retirement Planner**
    Estimates retirement corpus needed and years remaining to retirement.
